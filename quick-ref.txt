Files and Buffers
-----------------

C-x C-f                 Find file
C-x C-r                 find-file-read-only
C-x C-q                 read-only-mode
C-x RET r               revert-buffer-with-coding-system
C-x C-s                 Save buffer
C-x C-b                 List buffers
C-x b                   Select buffer
C-x k                   Kill buffer

C-g                     Quit (cancel)

Movement
--------

C-f                     Forward character
C-b                     Backward character
C-n                     Next line
C-p                     Previous line
C-a                     Beginning of line
C-e                     End of line
M-f                     Forward word
M-b                     Backward word
C-up, M-{               Backward paragraph
C-down, M-}             Forward paragraph
C-home, M-<             Beginning of buffer
C-end, M->              End of buffer

C-s                     Search forward
C-r                     Search backward
C-M-s                   Regexp search forward
C-M-r                   Regexp search backward
M-s .                   isearch-forward-symbol-at-point (C-s for next match)

M-x helm-imenu          Jump to function

Editing
-------

C-space                 Mark
M-w                     Copy region
C-w                     Kill region
C-d                     Delete character
C-k                     Kill line
C-y                     Yank
M-y                     Yank pop
C-/                     Undo

M-x comment-region      Comment region
M-x uncomment-region    Uncomment region

C-M-\                   Indent region

Windows
-------

C-x 0                   Delete window
C-x 1                   Delete other windows
C-x 2                   Split window below
C-x 3                   Split window right
C-x o, M-o              Other window
C-l                     Recenter top bottom

Help
----

C-h f                   Describe function
C-h k                   Describe key
C-h m                   Describe mode

Lisp
----

C-x C-e                 Eval last sexp

Helm
----

right, C-j              Go down a level
left, C-l               Go up one level

Projectile
----------

C-c p p                 Switch project
C-c p f                 Find file in project

Magit
-----

F10                     magit-status
